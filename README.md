# Portfolio Website

This is a static portfolio website for Naga Appalaneni's DevOps, Cloud, SRE, Platform Engineering, and AI Automation brand.

## Run Locally

Open `index.html` directly in a browser, or run:

```bash
python3 -m http.server 8080
```

Then open:

```text
http://localhost:8080
```

## Current Publish Notes

- LinkedIn and email links are populated.
- The resume page uses a working email request link until a final public PDF is selected.
- Project links point to verified public GitHub repositories.
- Certification content is a roadmap, not a claim that certifications are completed.
